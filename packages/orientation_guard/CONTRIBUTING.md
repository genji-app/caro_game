# CONTRIBUTING 🤝

We're excited you're interested in contributing to `orientation_guard`!

## How to Contribute

- **Report Bug**: Open an issue describing the platform (iOS, Android, Web), browser, and reproduction steps.
- **Suggestion**: Open an issue describing your idea.
- **Pull Request**: Fork the repository, create a branch, and submit a PR.

## Code Standards

1. **Linting**: Run `flutter analyze` and fix all warnings.
2. **Formatting**: Run `dart format .` before committing.
3. **Documentation**: All public APIs must have valid `dartdoc` comments in English.
4. **Tests**: Ensure all existing tests pass and add new tests if applicable.
